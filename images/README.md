# images/

Place your site image files here. The following files are required:

| File | Used for |
|------|----------|
| `mascot.gif` | Animated mascot shown on the welcome page hero and About page |
| `services.png` | Services page desktop icon and nav icon |
| `portfolio.png` | Portfolio page desktop icon and nav icon |
| `pricing.png` | Pricing page desktop icon and nav icon |
| `about.png` | About page desktop icon and nav icon |
| `contact.png` | Contact page desktop icon and nav icon |

PNG, JPG, GIF, or WebP formats all work. The mascot should be a GIF for animation.
