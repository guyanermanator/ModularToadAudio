# images/

Place your site image files here. The following files are required:

| File | Used for |
|------|----------|
| `ModularToadAudio.jpg` | Mascot shown on the welcome page hero and About page |
| `home.png` | Home page desktop icon and nav icon |
| `services.png` | Services page desktop icon and nav icon |
| `portfoilo.svg` | Portfolio page desktop icon and nav icon |
| `priceing.png` | Pricing page desktop icon and nav icon |
| `about.png` | About page desktop icon and nav icon |
| `contact.png` | Contact page desktop icon and nav icon |

PNG, JPG, GIF, or WebP formats all work.
